이 Flutter 프로젝트는 객체 조작을 위한 편리한 메서드를 제공하는 Subzero 플러그인의 사용법을 보여줍니다.

## 기능

이 예제는 Subzero 플러그인의 두 가지 주요 기능을 보여줍니다:

1. **copyWith 메서드**: 객체의 속성을 업데이트하여 새 인스턴스 생성
2. **toJson 메서드**: 객체를 JSON 형식으로 변환
